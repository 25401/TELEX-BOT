## TERMUX USER
```bash
$ pkg upgrade && pkg update
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/PikaBotz/Anya_v2-MD
$ cd Anya_v2-MD
$ npm i 
```

```bash

```

```bash
$ pkg install yarn -y

```
